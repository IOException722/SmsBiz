Functionality
1) Sending and receiving sms
2) Showing all the messages received by the user, even those that are received by the user before the app install.
3) Showing a notification whenever the user receives a sms.
4) The messages is grouped by the sender i.e. all the messages sent by one sender can be seen together
5) Full text searched implemented accordingly

Libraries: com.android.support:recyclerview-v7:23.0.1 (For RecyclerView)
